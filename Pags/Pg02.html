Gerais

  • *{height: 100%} = configuração global das CSS
  • margin:auto; = centralizado
  • height: 200px = altura da pagina
  • height: 100vh = alinhamento central a página
  • width: 600px; = largura da pagina
  • padding: 10px = acochoamento lateral
  • margin: auto = Autoregulação do cetralizamneto do texto
  • border: 10px = borda arredondada ou definida
  • box-shadow: 10px 10px 10px black> = sombra para o lado, sombra para baixo, sombrea vai
  • espalhar, transparencia(cor)
  • display:inline-block; = background travado
  • color = cor do texto
  • @font-face = fontes externas

Text

  • text-allign: justify = texto alinhado
  • text-allign: center = texto centralizado
  • text-allign: left/right = texto, esquerda ou direita
  • text-shadow: 1px 1px 2px = sombra do texto
  • text-decoration:underline = sublinhado
  • text-indent: 20px = inicio do paragrafo, identado
  • text-decoration:none; = tirar sublinhado do link ou text
  • font-size: 20px = tamanho da fonte
  • font-weight: bold = peso da fonte
  • font-style:normal, italic = estilo da fonte
  • font-family: arial = fonte da seleção escolhida

Pseudo classe -- estado de um elemento == ativo, marcado, vazio, habilitado

  • :Hover - flutuar

Pseudo Elemento -- conteúdo do elemento

  • ::after - content '[escrever algo]' --> antes
  • ::before - content 'conteúdo' --> depois
