Simples

  • p = paragrafo
  • hr = linha horizontal
  • br = quebra de linha
  • %#x = Emoji
  • address = endereço
  • strong = negrito semântico
  • em = Itálico semântico
  • mark style=cssParameter = marca texto
  • small = texto pequeno
  • del = texto riscado
  • ins = texto sublinhado
  • sup = sobrescrito em cima
  • sub = sobrescrito em baixo
  • code = fonte monoespaçada, Códigos-fonte
  • pre = identação respeitando os espaços
  • q = citação simples com aspas
  • blockquote cite="..." citação deslocada com link
  • abbr = abreviação
  • bdo = Texto invertido
  • span = cor no meio de um texto com classe
  • button = cria um botão clicavel

Listas

  • ol type=".." start=".." = Ordened list
  • li = lista
  • ul type=".." = lista não ordenada
  • dl = definition list (termo)
  • dd = description (descrição)

Links

  • rel="external" = link externo
  • rel="nofollow" não da aval ao conteudo
  • href="next" = link interno
  • href="prev" = link interno de retorno
  • target="_self" = link para a propria pagina
  • target="_blank" = Nova guia sem fechar a atual
  • ../ = voltar paginas fora da pasta
  • link rel="stylesheet" type="text/css" href="NameHere.css"> = CSS out pag

Downloads

  • a href="dowload/01-aula-curso-java-slides.zip"
  • type="dowload.zip" type="application/zip" Download ZIP = Download em zip ou pdf

Teclas de atalho

  • alt = troca de várias letras de uma vez
  • ctrl+ shift + p = envelope - wrap with abreviation
  • ctrl + mouse left click = criação de nova pagina
  • regra => import
  • seletor => body
  • declarações => font-family

Grouping tags

  • Header -> cabeça
  • nav -> links
  • main -> principal
  • section -> sessão
  • article -> artigo
  • aside -> periférico
  • footer -> rodapé
  • picture = varios sources de imagem
  • main = conteúdo principal
  • div = espaço
